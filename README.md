UpGrade is a Flutter-based graduation project designed to help students plan, manage, and improve their study habits using smart scheduling and AI-powered assistance.

---

##  Features

-  **Daily Planner**
  - View tasks in a clear daily timeline
  - Includes focus sessions and breaks

-  **Weekly Schedule**
  - Separate weekly schedule screen
  - Overview of all tasks grouped by day

-  **AI Chatbot**
  - Study guidance and productivity help

-  **Progress Dashboard**
  - Track productivity and study performance

-  **Burnout Risk Detection**
  - Monitor workload and stress indicators

-  **Group Study**
  - Collaborative study features

-  **Warnings & Notifications**
  - Important reminders and alerts

---

##  Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **Backend:** FastAPI (Python)
- **AI/LLM:** Llama 3.3 (70B) via Groq API
- **Platforms:** Android, iOS, Web, Windows
- **UI:** Material Design
- **Version Control:** Git & GitHub
